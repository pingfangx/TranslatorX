[125/173] RRakePopup(rake 弹窗) [汉化反馈]

To quickly execute a Rake task press &shortcut:org.jetbrains.plugins.ruby.tasks.rake.actions.RakeTasksPopupAction; or use Tools | Run Rake task and start typing the task name.

There's no need to start from the beginning of the task name. Just type mig to quickly find all migration tasks.
