RemoteDebug(远程调试) [汉化反馈]

有时会在一台机器上开发并在另一台机器上运行。有时你需要远程调试。CLion 支持使用 GDB/gdbserver 进行远程调试。这意味着在 gdbserver 下的一台机器上运行可执行文件，您可以使用另一台机器上的 CLion 的 GDB 连接它，并使用 CLion 调试器的所有有用的检查代码 UI：从 IDE 中设置断点，查看变量值，计算表达式等等。

要与 GDB 远程连接到目标主机，请创建 GDB 远程调试配置(转至 Run | Edit Configurations 并从模板创建它)，并提供远程连接设置，例如使用 GDB，介质(串行线路或使用 TCP 或 UDP 的 IP 网络)，调试符号，sysroot，路径映射：

要附加到在您的机器上本地运行而不是从 IDE 启动的进程，请选中 Run | Attach to Local Process...。
